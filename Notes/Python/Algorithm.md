## [并查集](https://blog.csdn.net/luomingjun12315/article/details/47373345)
需要实现的三个接口：
+ 建立新的集合
+ 将两个不相交集合进行合并，相交则不合并
+ 找到每个节点的代表节点，也可以判断两个节点是不是在一个集合内。
关键概念：节点的代表节点（即该节点的父节点），集合的代表节点（即该集合的根节点）
