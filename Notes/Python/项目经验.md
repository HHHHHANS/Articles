# Project Experience
## 闲置虚拟机预测
### 遇到的困难
+ tsFresh库内存泄漏问题：
	+ 数据清洗阶段发现电脑内存释放不及时的问题：线性增长；
	抽取出单次任务的代码进行time.sleep，强制释放内存。
		
	+ 使用内存监测插件，找出内存泄漏模块、泄漏代码：构建序列特征时候某些特征含有的子特征过大，优化不好，重新评估特征重要性，跳过无用特征构建，问题解决。
